Problem 1: The Rolling Die & Number Theory

Scenario: A fair 6-sided die is rolled once. Let S be the sample space. Let A be the event of getting a prime number, B be the event of getting an even number, and C be the event of getting a number greater than 4.

1. List the elements of S and find n(S).

2. List elements of A, B, and C.

3. Find P(A), P(B), and P(C).

4. Find P(A ∩ B).

5. Find P(A ∪ B).

6. Are A and B mutually exclusive? Why?

7. Find P(A | C).

8. Are A and C independent?

9. Find P(B ∩ C').

10. Find P(A ∪ B ∪ C).

Model Answer:

  • 1. S={1,2,3,4,5,6}, n(S)=6
  • 2. A={2,3,5}, B={2,4,6}, C={5,6}
  • 3. P(A)=3/6=0.5, P(B)=3/6=0.5, P(C)=2/6=1/3
  • 4. A∩B={2} -> P(A∩B)=1/6
  • 5. P(A∪B) = 0.5 + 0.5 - 1/6 = 5/6
  • 6. No, A∩B={2} ≠ ∅ (Not disjoint)
  • 7. P(A∩C)={5}=1/6. P(A|C)=(1/6)/(1/3)=0.5
  • 8. Check P(A∩C)=1/6 vs P(A)P(C)=0.5*(1/3)=1/6. Yes, Independent.
  • 9. B∩C' = {2,4,6} ∩ {1,2,3,4} = {2,4} -> P=2/6=1/3
  • 10. A∪B∪C = {2,3,4,5,6} -> P=5/6
الشرح بالعربي:
الرقم 1 ليس أولياً، لذا A تضم {2,3,5}. الأحداث المتنافية لازم يكون تقاطعها فاي، وهنا عندنا الـ 2 مشترك فبالتالي مش متنافيين. عشان نثبت الاستقلال بنضرب الاحتمالات المنفردة لو ساوت التقاطع يبقى مستقلين.

Problem 2: Conditional Probability in Production

Scenario: A factory has two machines. Machine 1 (M1) produces 60% of items, Machine 2 (M2) produces 40%. 5% of M1's items are defective (D), while 10% of M2's items are defective.

1. Define P(M1) and P(M2).

2. Define P(D | M1) and P(D | M2).

3. Find P(M1 ∩ D).

4. Find P(M2 ∩ D).

5. Find the total probability P(D).

6. Find P(D').

7. If an item is defective, what is the probability it came from M1? P(M1 | D).

8. Find P(M2 | D).

9. Are "Being Defective" and "Machine 1" independent?

10. What is P(M1 ∪ D)?

Model Answer:

  • 1. P(M1)=0.6, P(M2)=0.4
  • 2. P(D|M1)=0.05, P(D|M2)=0.10
  • 3. P(M1∩D) = 0.6 * 0.05 = 0.03
  • 4. P(M2∩D) = 0.4 * 0.10 = 0.04
  • 5. P(D) = 0.03 + 0.04 = 0.07
  • 6. P(D') = 1 - 0.07 = 0.93
  • 7. P(M1|D) = 0.03 / 0.07 = 3/7 ≈ 0.428
  • 8. P(M2|D) = 0.04 / 0.07 = 4/7 ≈ 0.571
  • 9. No, P(D|M1) (0.05) ≠ P(D) (0.07).
  • 10. P(M1∪D) = 0.6 + 0.07 - 0.03 = 0.64
الشرح بالعربي:
دي مسألة "Total Probability" و "Bayes Rule". بنجيب التقاطعات الأول بضرب الاحتمال الشرطي في احتمال الماكينة. عشان نجيب P(D) الكلية بنجمع التقاطعات. ولما يطلب احتمال الماكينة بشرط العيب (P(M1|D)) بنقسم الجزء على الكل.

Problem 3: Algebra of Events (Laws Only)

Scenario: For two events A and B, we have P(A) = 0.5, P(B) = 0.6, and P(A ∩ B) = 0.2.

1. Find P(A ∪ B).

2. Find P(A').

3. Find P(A ∩ B').

4. Find P(B ∩ A').

5. Find P(A' ∩ B').

6. Find P(A' ∪ B').

7. Find P(A | B).

8. Find P(B | A).

9. Find P(A' | B').

10. Are A and B independent? Justify.

Model Answer:

  • 1. P(A∪B) = 0.5+0.6-0.2 = 0.9
  • 2. P(A') = 1-0.5 = 0.5
  • 3. P(A∩B') = P(A)-P(A∩B) = 0.5-0.2 = 0.3
  • 4. P(B∩A') = P(B)-P(A∩B) = 0.6-0.2 = 0.4
  • 5. P(A'∩B') = 1-P(A∪B) = 1-0.9 = 0.1 (De Morgan)
  • 6. P(A'∪B') = 1-P(A∩B) = 1-0.2 = 0.8
  • 7. P(A|B) = 0.2 / 0.6 = 1/3 ≈ 0.33
  • 8. P(B|A) = 0.2 / 0.5 = 0.4
  • 9. P(A'|B') = P(A'∩B')/P(B') = 0.1 / 0.4 = 0.25
  • 10. P(A)P(B) = 0.5*0.6 = 0.3. Since 0.3 ≠ 0.2, they are Dependent.
الشرح بالعربي:
دي تطبيق مباشر للقوانين. لاحظ قانون دي مورجان: (A'∩B') هو "واحد ناقص الاتحاد". وقانون الفرق (P(A∩B')) هو "الأول ناقص التقاطع". الاستقلال لازم حاصل الضرب يساوي التقاطع بالظبط.

Problem 4: Tossing a Coin 3 Times

Scenario: A fair coin is tossed 3 times. Let E1 = at least two heads, E2 = exactly one tail, E3 = the first toss is a head.

1. Find n(S).

2. List elements of E1.

3. List elements of E2.

4. List elements of E3.

5. Find P(E1) and P(E2).

6. Find E1 ∩ E2 and its probability.

7. Find P(E1 | E3).

8. Find P(E2 | E3).

9. Are E2 and E3 independent?

10. Find P(E1 ∪ E2).

Model Answer:

  • 1. n(S) = 2^3 = 8
  • 2. E1 = {HHH, HHT, HTH, THH}
  • 3. E2 = {HHT, HTH, THH} (Note: Exactly 1 T = 2 H)
  • 4. E3 = {HHH, HHT, HTH, HTT}
  • 5. P(E1)=4/8=0.5, P(E2)=3/8
  • 6. E1∩E2 = {HHT, HTH, THH} -> P=3/8
  • 7. E1∩E3 = {HHH, HHT, HTH} -> P=3/8. P(E1|E3) = (3/8)/(4/8) = 0.75
  • 8. E2∩E3 = {HHT, HTH} -> P=2/8. P(E2|E3) = (2/8)/(4/8) = 0.5
  • 9. P(E2)P(E3) = (3/8)*(1/2) = 3/16. P(E2∩E3)=2/8=4/16. No, Dependent.
  • 10. P(E1∪E2) = 4/8 + 3/8 - 3/8 = 4/8 = 0.5
الشرح بالعربي:
هنا لازم نكتب فضاء العينة صح (8 نواتج). كلمة "At least 2 heads" يعني يا اتنين يا تلاتة. لاحظ إن E2 هي جزء من E1 (subset) لأن اللي عنده كتابه واحدة بالظبط أكيد عنده صورتين. عشان كدة الاتحاد طلع هو هو E1.

Problem 5: Special Relationships (Subset & Disjoint)

Scenario: Suppose A and B are events where P(A) = 0.3 and P(B) = 0.5. Calculate the requested values for Two Different Cases.

Case 1: A and B are Mutually Exclusive
Case 2: A is a Subset of B (A ⊂ B)

1. Find P(A ∩ B) in Case 1.

2. Find P(A ∪ B) in Case 1.

3. Find P(A | B) in Case 1.

4. Find P(A' ∩ B) in Case 1.

5. Find P(A' ∩ B') in Case 1.

6. Find P(A ∩ B) in Case 2.

7. Find P(A ∪ B) in Case 2.

8. Find P(B | A) in Case 2.

9. Find P(A | B) in Case 2.

10. Find P(B ∩ A') in Case 2.

Model Answer:

  • Case 1:
  • 1. P(A∩B) = 0 (Disjoint)
  • 2. P(A∪B) = 0.3 + 0.5 = 0.8
  • 3. P(A|B) = 0 / 0.5 = 0
  • 4. P(A'∩B) = P(B) - 0 = 0.5
  • 5. P(A'∩B') = 1 - 0.8 = 0.2
  • Case 2:
  • 6. P(A∩B) = P(A) = 0.3
  • 7. P(A∪B) = P(B) = 0.5
  • 8. P(B|A) = 0.3 / 0.3 = 1
  • 9. P(A|B) = 0.3 / 0.5 = 0.6
  • 10. P(B∩A') = P(B) - P(A) = 0.5 - 0.3 = 0.2
الشرح بالعربي:
دي أهم مسألة في الامتحان!
1- لو "Mutually Exclusive" يعني التقاطع بصفر فوراً.
2- لو "A subset of B" يعني الصغير جوه الكبير، يبقى التقاطع هو الصغير (A) والاتحاد هو الكبير (B). لاحظ إن P(B|A) في حالة الـ subset دايما بـ 1 لأننا متأكدين إن لو الصغير حصل يبقى الكبير حصل بالتبعية.
